Class: StreamedBy

StreamedBy(baseUrl, manifestPath, videoElem) → {Promise.<string>}

new StreamedBy(baseUrl, manifestPath, videoElem) → {Promise.<string>}

Sets the source of a video element to a stream. Fallbacks to HLS when DASH is not possible. Uses DashPlayer internally.
Parameters:
Name Type Description
baseUrl string the url to the folder containing the manifest and chunk files
manifestPath string the filename of the .mpd/.m3u8 manifests without file extensions
videoElem HTMLVideoElement the video element that should play the stream
